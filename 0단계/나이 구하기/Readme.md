# 🧮 문제 설명

머쓱이는 선생님이 몇 년도에 태어났는지 궁금해졌습니다.  
2022년 기준 선생님의 나이 `age`가 주어질 때,  
선생님의 **출생 연도**를 `return` 하는 `solution` 함수를 완성해주세요.

## 🔒 제한사항
- `0 < age ≤ 120`
- 나이는 **태어난 연도에 1살**이며  
  매년 **1월 1일마다 1살씩 증가**합니다.

## 📥 입출력 예

| age | result |
|-----|---------|
| 40  | 1983    |
| 23  | 2000    |

## 📘 입출력 예 설명

### 예시 1
- 2022년 기준 나이: 40  
- `2022 - 40 + 1 = 1983`  
- → `return 1983`

### 예시 2
- 2022년 기준 나이: 23  
- `2022 - 23 + 1 = 2000`  
- → `return 2000`

> ※ 2024년 3월 14일 지문이 더 명확하게 수정되었습니다.
