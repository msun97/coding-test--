# 🧮 문제 설명

정수 `num1`, `num2`가 매개변수로 주어질 때,  
`num1`을 `num2`로 나눈 **나머지**를 `return` 하도록 `solution` 함수를 완성해주세요.

## 🔒 제한사항
- `0 < num1 ≤ 100`
- `0 < num2 ≤ 100`

## 📥 입출력 예

| num1 | num2 | result |
|------|------|---------|
| 3    | 2    | 1       |
| 10   | 5    | 0       |

## 📘 입출력 예 설명

### 예시 1
- `num1 = 3`, `num2 = 2`  
- `3 % 2 = 1`  
- → `return 1`

### 예시 2
- `num1 = 10`, `num2 = 5`  
- `10 % 5 = 0`  
- → `return 0`
