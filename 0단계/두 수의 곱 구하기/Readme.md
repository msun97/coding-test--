# 🧮 문제 설명

정수 `num1`, `num2`가 매개변수로 주어집니다.  
`num1`과 `num2`를 **곱한 값**을 `return` 하도록 `solution` 함수를 완성해주세요.

## 🔒 제한사항
- `0 ≤ num1 ≤ 100`
- `0 ≤ num2 ≤ 100`

## 📥 입출력 예

| num1 | num2 | result |
|------|------|---------|
| 3    | 4    | 12      |
| 27   | 19   | 513     |

## 📘 입출력 예 설명

### 예시 1  
- `num1 = 3`, `num2 = 4`  
- `3 * 4 = 12` → `return 12`

### 예시 2  
- `num1 = 27`, `num2 = 19`  
- `27 * 19 = 513` → `return 513`
