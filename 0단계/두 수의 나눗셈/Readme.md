# 🧮 문제 설명

정수 `num1`과 `num2`가 매개변수로 주어질 때,  
`num1`을 `num2`로 나눈 값에 **1,000을 곱한 후 정수 부분**을 `return` 하도록 `solution` 함수를 완성해주세요.

## 🔒 제한사항
- `0 < num1 ≤ 100`
- `0 < num2 ≤ 100`

## 📥 입출력 예

| num1 | num2 | result |
|------|------|---------|
| 3    | 2    | 1500    |
| 7    | 3    | 2333    |
| 1    | 16   | 62      |

## 📘 입출력 예 설명

### 예시 1
- `num1 = 3`, `num2 = 2`  
- `3 / 2 = 1.5`  
- `1.5 * 1000 = 1500`  
- → `return 1500`

### 예시 2
- `num1 = 7`, `num2 = 3`  
- `7 / 3 = 2.33333...`  
- `* 1000 = 2333.333...`  
- 정수 부분 → `2333`

### 예시 3
- `num1 = 1`, `num2 = 16`  
- `1 / 16 = 0.0625`  
- `* 1000 = 62.5`  
- 정수 부분 → `62`
