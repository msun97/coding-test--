# 문제 설명
정수 `num1`과 `num2`가 매개변수로 주어집니다.  
두 수가 **같으면 1**, **다르면 -1**을 `return` 하도록 `solution` 함수를 완성해주세요.

## 제한사항
- `0 ≤ num1 ≤ 10,000`
- `0 ≤ num2 ≤ 10,000`

## 입출력 예

| num1 | num2 | result |
|------|------|--------|
| 2    | 3    | -1     |
| 11   | 11   | 1      |
| 7    | 99   | -1     |

## 입출력 예 설명

### 예시 1  
- `num1 = 2`, `num2 = 3`  
- 두 수는 다르므로 `-1`을 return합니다.

### 예시 2  
- `num1 = 11`, `num2 = 11`  
- 두 수는 같으므로 `1`을 return합니다.

### 예시 3  
- `num1 = 7`, `num2 = 99`  
- 두 수는 다르므로 `-1`을 return합니다.
